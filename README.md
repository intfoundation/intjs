# Intjs

This is the INT JavaScript API. It's avaiable on npm as a node module.


## Installation

### Node.js

```bash
npm install intjs
```

### Usage

```bash
let Intjs = require('intjs');
let intjs = new Intjs(rpcHost, rpcPort);
```

## Dependencies

* Node.js
* npm
